This project is based on web scrapping of an e-commerce website (etsy.com)

installing.py is the first step of importing all required modules

testcase1.py includes search bar query test case

testcase2.py includes the link text clicking test case

full_testing_etsy.py includes all the functionality testing of the website including login, add to basket, buy, text link, search query, scroll up and scroll down pages etc.
